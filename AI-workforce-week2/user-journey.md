## User Journey in Microsoft 365

In Microsoft 365, creating a user account is typically done by an administrator through the Microsoft 365 admin center, where the admin enters basic details such as the user’s name, username, display name, and location.

After the account is created, the system generates a sign-in identity that allows the new user to securely access organizational services using a unique email address and password.

The administrator can then assign roles if needed, depending on whether the user requires standard access or elevated administrative permissions.

Once the account exists, the next important step is assigning a Microsoft 365 license to the user.

License assignment is completed from the same admin portal by selecting the appropriate license plan and enabling the services included in that plan for the user.

Assigning a license activates access to core productivity applications such as Word, Excel, PowerPoint, Outlook, and collaboration tools like Teams and SharePoint.

Without a license, the user account exists but cannot actively use most Microsoft 365 services.

Licensing also enables access to OneDrive cloud storage, allowing users to securely save and retrieve files across multiple devices.

Another important benefit of license assignment is access to organizational email services, which allows users to send and receive messages using their official work identity.

Licenses also help administrators control which tools and services each user can access based on job responsibilities.

This improves security by ensuring users only interact with resources relevant to their roles.

License assignment supports compliance and monitoring because administrators can track service usage across the organization.

It also helps organizations manage subscription costs by allocating resources efficiently.

Overall, creating a user account and assigning a license ensures that each individual can securely access the right tools needed to communicate, collaborate, and remain productive within the Microsoft 365 environment.
